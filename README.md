# MOFS
This repository contains the official implementation codes of **Multi-Operator Few-Shot Learning for Generalization Across PDE Families**.


## Datasets
The dataset can be downloaded from [Google Drive](https://drive.google.com/drive/u/2/folders/1DsCBr2EDT1fgPjZYfiRuG-K-mpE-aqwm).


## Run Experiments
Each script file can be run by "python 'file_name'".

```bash
python train_2D_DarcyFlow_beta10.0.py
```
